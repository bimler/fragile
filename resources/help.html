Fragile Help Page

Welcome to the Fragile help page, below you can find instructions on how to use the Fragile Software.

Entering Your Fraction

Soft Keys

+/-

This softkey can be used to change the sign of the fraction in focus.

C

Clears the input field.

<--

Will backspace the current part of the fraction in focus and move the focus backward if empty.

+, -, x, /

These are operation keys to perform operations between two fractions.

=

Displays the result of the operation between the two fractions.

R

Resets input and output data.

Inv

Inverts the fraction in focus.

Hist>

Displays the history of operations.

^

Multiplies number by the power of another number.

Focus

A yellow highlight around the number determines where the focus is. When the user is focused on a number, he or she can enter a new number (if the number is 0) or press the delete/backspace soft key to delete the highlighted number. Shifting focus can be used by pressing the period (".") or comma (",") keys on his or her physical keyboard or by pressing the right arrow soft key (">"). 

Viewing Results

Input Field

Where the user will enter the two numbers to be calculated. Once the calculation is completed, the input field will display the answer.

History Field

Displays the history of every calculation the user has made before closing the calculator.

Intermediate Steps

The steps taken to complete a calculation.

Menu

File

Print

Prints the history of the calculations.

Playback

Plays the recorded history back.

Record

Saves the history of the calculations to a file.

Exit

Exits the application.

Display

Style

Choose from 3 different kinds of ways to display the bar of fractions. The 3 ways are Bar, Slash, and Solidus.

Mode

Choose from 3 different ways to to represent fractons. The 3 ways are Mixed, Improper, and Proper.

Pie Chart

Shows a representation of the result in a pie chart.

Help

About

Information about the product version and the user agreement.

Open in Browser

Opens the online help page for the Fragile Calculator.

 Farey

Given a fraction, the Farey button (denoted by gamma) returns the next fraction in the Farey sequence. A Farey sequence includes every fraction in its most simplified form up to the given denominator.
